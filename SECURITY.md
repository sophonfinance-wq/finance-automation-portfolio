# Security Policy

## Data

This repository contains **only fully synthetic data**. There are no real client, employer, or
personal records; no credentials; and no secrets committed. Every entity, account, and figure is
invented for demonstration.

## Reporting a concern

If you find a security issue — for example a dependency advisory or an unintended data exposure —
please report it privately by email to **sophonfinance@gmail.com** rather than opening a public
issue. You can expect an acknowledgement within a few business days.

## Supported version

The latest release on the `main` branch is the supported version. Dependencies are monitored
automatically via Dependabot (`.github/dependabot.yml`).
