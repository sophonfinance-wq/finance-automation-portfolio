Self-healing operations

The engines run the control loop themselves.

Each engine watches its own output, corrects routine drift from source data, re-verifies every correction against an independent recomputation, and escalates the judgment calls to a person. Nothing self-certifies.

SOURCEGL · bank · schedules
COMPUTEFinance engine
SEPARATE LOGICIndependent recompute
DECISIONTie-out
Ties → posts with evidenceWhen the independent check confirms the work, it posts with a written audit trail.
Routine break → fixed from sourceA mis-keyed cell or dropped intercompany leg is corrected, then re-verified before it counts.
Judgment call → a personWhat the loop cannot prove is quarantined and flagged — never rubber-stamped.

You set the policy: full autonomy in approved environments, or a human sign-off on every post — the controls and audit trail are identical either way. Live today on five engines; every loop writes its own run report, with sample reports committed on GitHub.
